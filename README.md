# MyCybersecurityCollection
Lightweight cybersecurity and networking tools that can run even on old Android devices.

## 🔧 Features

- TCP Port Scanner
- Banner Grabbing
- HTTP Header Analysis
- Recursive URL Extractor (with depth limit)

## 🧪 Testing Environment
Tested on:
    - Lenovo A2020a40 (Android 5.1.1, QPython 3H)
    - Huawei MediaPad T5 (Android 8, EMUI 8, Pydroid 3)
    - Redmi Note 11 (Android 12, Pydroid 3 Premium)
    - Samsung Galaxy A04s (Android 13, One UI Core 5.0, Pydroid 3 Premium)

It’s practically impossible to say that you can’t run it on your android device.
